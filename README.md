# mk-demo

该demo是使用mk框架创建的一个demo

[在线效果请点这里](https://ziaochina.github.io/mk-demo/)

## 环境准备

- nodejs最新版
- npm最新版
- chrome

## start

```
$ npm install
$ npm start
http://127.0.0.1:8089
```

## 注意

- 如果npm start报错not found module
```
$ npm cache clean
$ rm -rf node_modules
$ rm -rf package-lock.json
$ npm install
```
